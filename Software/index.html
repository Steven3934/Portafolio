SOFTWARE

Sistemas operativos

Windows

Windows es el nombre de una familia de distribuciones de software para PC, servidores, sistemas empotrados y antiguamente teléfonos inteligentes desarrollados y vendidos por Microsoft y disponibles para múltiples arquitecturas, tales como x86, x86-64 y ARM

Linux

GNU/Linux es el término semánticamente correcto para referirse a la familia de Sistemas Operativos de código abierto basados en UNIX. Esta familia de sistemas operativos se caracterizan por ser multiplataforma, multitarea y multiusuario además de presentar una arquitectura diferente a los sistemas anteriormente mencionados que hacen que un computador trabaje y maneje sus recursos de hardware de la forma más eficiente posible.

Mac

macOS es una serie de sistemas operativos gráficos desarrollados y comercializados por Apple desde 2001. Es el sistema operativo principal para la familia de computadoras Macintosh de Apple.

Sistemas operativos

Un sistema operativo es el conjunto de programas de un sistema informático que gestiona los recursos del hardware y provee servicios a los programas de aplicación de software. Estos programas se ejecutan en modo privilegiado respecto de los restantes.

Sistemas operativos moviles

Android

Android es un sistema operativo basado en núcleo Linux, y utiliza una arquitectura de software de capas. La capa superior es la interfaz de usuario, donde los usuarios interactúan con el dispositivo.

iOS

iOS es un sistema operativo móvil de código cerrado desarrollado por Apple Inc. Originalmente desarrollado para el iPhone, después se utilizó en dispositivos como el iPod touch y el iPad.

Software libre

El software libre es aquel que respeta la libertad de los usuarios y la comunidad en general. Esto significa que los usuarios tienen la libertad de ejecutar, copiar, distribuir, estudiar, modificar y mejorar el software. Para garantizar estas libertades, el software libre suele distribuirse con su código fuente, permitiendo así que cualquiera lo estudie y lo modifique según sus necesidades con algunos ejemplos como: Linux, VLC Media player, Mozilla Firefox... etc

Herramientas del sistema

Comprobador de errores

La herramienta de Comprobación de errores en el disco (Scandisk) busca posibles errores en el disco e intenta repararlos. Podemos utilizarla sobre cualquier medio de almacenamiento (discos duros, discos externos, memorias flash, etc.), excepto con los de sólo lectura (CD's y DVD's).

Desfragmentador

Cuando se graban archivos en un disco vacío, se guardan de una pieza y uno a continuación de otro. Pero con el tiempo, al borrar y modificar archivos, van quedando huecos entre ellos. Windows dividirá en varios fragmentos los nuevos archivos y los irá colocando en el espacio libre.

Información del Sistema

Esta herramienta como su propio nombre indica se encarga de mostrar información sobre el sistema, como por ejemplo, sobre las aplicaciones instaladas, los controladores, el procesador, etc.

Restaurar sistema

Esta herramienta se encarga de restaurar el sistema tal cual estaba anteriormente. Para ello dispone de puntos de restauración, es decir, momentos en que el sistema se hizo una copia del estado del registro de Windows y que podemos utilizar para volver a un estado anterior.

Comandos D.O.S

En Windows puedes introducir comandos MS-DOS usando el símbolo del sistema de Windows. El intérprete de línea de comandos “cmd.exe” utiliza comandos CMD de Windows que contienen comandos DOS. El intérprete de líneas de comando también contiene varias características del intérprete de DOS COMMAND.COM que antes faltaban.

Instalacion

Windows

Para instalar Windows, primero descarga la herramienta de creación de medios desde el sitio web oficial de Microsoft y crea un USB de arranque con la imagen de Windows. Conecta el USB a tu computadora y reinicia, asegurándote de que el BIOS/UEFI esté configurado para arrancar desde el USB. Sigue las instrucciones en pantalla para seleccionar el idioma, la edición de Windows y la arquitectura (32 o 64 bits). Acepta los términos de licencia, elige "Personalizada: Instalar solo Windows (avanzado)", selecciona o crea una partición donde instalar el sistema operativo y sigue las indicaciones para completar la instalación. Una vez finalizada, el sistema se reiniciará varias veces; después, configura tus preferencias y cuentas de usuario según se te solicite.

Linux

Para instalar Linux, primero descarga la imagen ISO de tu distribución preferida desde su sitio web oficial y crea un USB de arranque utilizando una herramienta como Rufus o Etcher. Conecta el USB a tu computadora y reiníciala, configurando el BIOS/UEFI para que arranque desde el USB. Una vez que el sistema arranque desde el USB, selecciona "Instalar" en el menú de inicio. Sigue las instrucciones en pantalla para elegir el idioma, la zona horaria y la distribución del teclado. Luego, selecciona el disco o partición donde quieres instalar Linux, elige si deseas realizar una instalación limpia o dual boot, y proporciona los detalles de tu cuenta de usuario. Después de confirmar los cambios, la instalación comenzará y, al finalizar, reinicia tu computadora y retira el USB de arranque para completar el proceso de instalación.

Software aplicativo (Word)

Para instalar Microsoft Word, primero necesitas adquirir una licencia del software, ya sea comprándolo como parte de la suite de Microsoft Office o de forma independiente. Una vez que tengas la licencia, puedes descargar el instalador desde el sitio web oficial de Microsoft o utilizar un disco de instalación si lo tienes. Ejecuta el instalador y sigue las instrucciones en pantalla, aceptando los términos de uso y seleccionando la ubicación de instalación si es necesario. Una vez completada la instalación, puedes abrir Word desde el menú de inicio o desde el acceso directo en tu escritorio y activarlo utilizando la clave de producto que te proporcionaron con la licencia.

Antivirus (Avast)

Para instalar Avast antivirus, primero descarga el instalador desde el sitio web oficial de Avast. Una vez descargado, haz doble clic en el archivo para ejecutarlo. Sigue las instrucciones en pantalla para completar la instalación, aceptando los términos de uso y seleccionando las opciones de configuración según tus preferencias. Una vez instalado, Avast realizará automáticamente una actualización de la base de datos de virus y estará listo para proteger tu dispositivo contra amenazas en línea y malware.

El software de aplicación es una categoría de programas informáticos diseñados con el propósito de facilitar a los usuarios la realización de determinadas tareas, es decir, como verdaderas herramientas de trabajo. Por ejemplo: Microsoft Word, Google Chrome, Adobe Photoshop, Avast.

Lenguajes de programacion

Java Script

JavaScript es un lenguaje de programación esencial en el desarrollo web, permitiendo la creación de sitios dinámicos y aplicaciones interactivas. Su capacidad para ejecutarse en el navegador del usuario lo hace fundamental para añadir funcionalidades como validación de formularios, efectos visuales y manipulación de contenido en tiempo real. Es uno de los lenguajes más populares y ampliamente utilizados en la industria del desarrollo de software, siendo fundamental para una gran cantidad de desarrolladores web y de aplicaciones.

Python

Python es un lenguaje de programación versátil y de alto nivel, reconocido por su sintaxis clara y legible. Es utilizado en una amplia variedad de aplicaciones, desde desarrollo web y científico hasta automatización de tareas y desarrollo de juegos. Su popularidad ha crecido considerablemente en los últimos años gracias a su facilidad de aprendizaje y su comunidad activa de desarrolladores. Python es una opción preferida para aquellos que buscan eficiencia y productividad en el desarrollo de software, y su uso sigue en aumento en diversos campos de la informática y más allá.

Java

Java es un lenguaje de programación de propósito general y orientado a objetos que se destaca por su portabilidad y robustez. Ampliamente utilizado en el desarrollo de aplicaciones empresariales, móviles y web, Java es conocido por su sintaxis clara y su amplia gama de bibliotecas y frameworks. Su capacidad para ejecutarse en cualquier plataforma que tenga instalada la máquina virtual de Java (JVM) lo convierte en una opción popular para aplicaciones multiplataforma. Java sigue siendo uno de los lenguajes de programación más utilizados en la actualidad, especialmente en el ámbito empresarial y académico.

C#

C# es un lenguaje de programación desarrollado por Microsoft que se destaca por su potencia y facilidad de uso. Diseñado especialmente para el desarrollo de aplicaciones en el entorno de .NET Framework y .NET Core, C# es ampliamente utilizado en el desarrollo de aplicaciones de escritorio, aplicaciones web, juegos y aplicaciones móviles. Su sintaxis similar a la de otros lenguajes como Java y C++ lo hace accesible para los desarrolladores, mientras que su integración con las herramientas de desarrollo de Microsoft, como Visual Studio, lo convierte en una opción atractiva para los profesionales de la industria. C# continúa siendo una opción popular entre los desarrolladores, especialmente en el ecosistema de desarrollo de software de Microsoft.

Gestores de descarga

Los gestores de descarga son programas diseñados para facilitar y mejorar el proceso de descarga de archivos desde internet. Estos programas permiten a los usuarios descargar archivos más rápidamente, gestionar varias descargas simultáneamente, y reanudar descargas interrumpidas sin perder el progreso. Además, suelen ofrecer características avanzadas como la programación de descargas, la segmentación de archivos para descargas más rápidas y la organización de archivos descargados. Los gestores de descarga son especialmente útiles para manejar archivos grandes o múltiples descargas, mejorando la eficiencia y la experiencia del usuario al descargar contenido de la web. Ejemplos populares de gestores de descarga incluyen Internet Download Manager (IDM), JDownloader y Free Download Manager (FDM).
